# Code of Conduct

## Our pledge

We want this project to be a welcoming, technically serious place to discuss awareness protocols, adapters, and governance-adjacent tooling.

## Expected behavior

- Be respectful and concise.
- Critique ideas, not people.
- Keep protocol discussions concrete and evidence-based.
- Respect scope guardrails and proprietary boundaries.

## Unacceptable behavior

- Harassment or personal attacks
- Bad-faith derailment
- Publishing private implementation details that are explicitly out of scope
- Turning protocol review into product-pitch spam

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict participation when behavior violates this code.
