SIGNAL CONTRACT

Awareness events for agents, data systems, and devices.

Signal Contract standardizes a small JSON object that producers emit and renderers consume.

One JSON object. Many renderers. No shared backend required.

What it is

Signal Contract is the awareness layer. It does not govern what agents may publish. It standardizes how systems emit awareness when meaningful state changes occur.

Constitutional CMS governs what agents are permitted to publish. Signal Contract standardizes how systems emit awareness when governed state changes.
