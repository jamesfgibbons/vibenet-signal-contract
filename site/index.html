VIBENET SIGNAL CONTRACT

Flat awareness events for agents, systems, and devices.

VIBEnet Signal Contract standardizes one small JSON object that producers emit and renderers consume.

One JSON object. Many renderers. No shared backend required.

What it is

Signal Contract is the open protocol package behind the public reference surface at vibenet.ai/protocol.

It is served live from VIBEnet, published here under the creator’s GitHub account, and kept aligned with the canonical schema URL at https://vibenet.ai/protocol/v1/schema.json.

Canonical example

  "pulse": 0.8
